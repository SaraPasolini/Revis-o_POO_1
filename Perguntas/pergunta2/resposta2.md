# 2. O que é uma classe e um objeto? Qual a relação entre eles? Dê um exemplo simples.

- A programação orientada a objetos é um paradigma de programação que é modelada com base no conceito de classes e seus relacionamentos. Organiza o código em torno de objetos em vez de funções ou lógica.

- Vantagens da POO :
1. Abstração: ocorre por meio da definição de estruturas de dados que encapsulam não apenas os atributos, mas também as funções e métodos responsáveis por manipulá-los.

2. Reutilização de código: baseia-se na possibilidade de aproveitar soluções já desenvolvidas. Muitas vezes, um software pronto precisa apenas de pequenas adaptações para resolver novos problemas ou adicionar
funcionalidades.
